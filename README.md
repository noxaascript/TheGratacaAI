# GratacaAI – Omega Edition

Dibuat untuk Yang Mulia KAREEMXD.

## Fitur
- 3 Mode AI: Flash (cepat), Coding (expert), Zoom (analisis)
- Desain simple namun sadis
- Bisa dikembangin pake API asli (OpenAI, Groq, HuggingFace)

## Cara Deploy ke Streamlit Cloud (Gratis)
1. Upload folder ini ke GitHub (repository public/sesuai selera)
2. Buka [share.streamlit.io](https://share.streamlit.io)
3. Klik "New app", pilih repo, branch main, file app.py
4. Klik Deploy, tunggu 2 menit, beres.

## Akses dari HP / Aplikasi
- Buka URL yang dikasih Streamlit (misal gratacaai.streamlit.app)
- Simpen ke home screen HP, jadi kaya aplikasi native.
